# PickAI - AI 智能选品工具

## 项目简介
PickAI 帮助跨境电商卖家快速找到高利润产品。

## 核心功能
- AI 驱动的选品分析
- 利润率计算
- 市场趋势预测

## 技术栈
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- AI: OpenAI API

## 快速开始
\`\`\`bash
npm install
npm run dev
\`\`\`
